# CPSC 335 - Project 3

   **Overview**: Implementation of Project 3 for CPSC-335:

   * Algorithm 1 - Skyline Visibility: Program that takes an array as input and outputs two lists containing the indices of buildings visiible from the left and right.

   * Algorithm 2 - Activity Selection: Program that uses a list of tuples activities as input to output a list of activities that maximize non-overlapping participation.

   * Programming language used for both algorithms: C++.
