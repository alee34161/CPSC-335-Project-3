# CPSC 335 - Project 3

   **Overview**: Implementation of Project 3 for CPSC-335:

   * Algorithm 1 - Skyline Visibility: Program that takes an array as input and outputs two lists containing the indices of buildings visiible from the left and right.

   * Algorithm 2 - Activity Selection: Program that uses a list of tuples activities as input to output a list of activities that maximize non-overlapping participation.

   * Programming language used for both algorithms: C++.

## Execution:

   1. Algorithm 1 program takes input from `input1.txt` and places output in `output1.txt`.

      * **Windows**: Download files and compile in a code editor. Navigate on and click to `output1.txt` to view output.

      * **Linux**: On console type `./algo1.out` then `cat output1.txt` to view output.

   2. Algorithm 2 program takes input from `input2.txt` and places output on `output2.txt`.

      * **Windows**: Download files and compile in a code editor. Navigate to and click on `output2.txt` to view output.
